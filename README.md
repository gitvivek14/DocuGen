# DocuGen

Privacy-first document generators built with Next.js. Users can fill, preview, and download clean PDFs without signup or server-side document storage.

## Tech Stack

- Next.js 16
- React 18
- TypeScript
- Tailwind CSS
- React Hook Form and Zod
- jsPDF

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` to view the app.

## Scripts

```bash
npm run lint
npm run typecheck
npm run build
```

## Repository Notes

This project uses npm. Commit `package-lock.json` and do not commit `node_modules`, `.next`, local environment files, or deployment output.
